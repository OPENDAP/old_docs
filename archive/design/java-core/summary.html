DODS Java Client-Side API Summary

The difierences between this and the C++ implementation

A paper describes the differences between this implementation of the DODS DAP and the C++ implementation. The later is described by the programmer's Guide on the DODS web page. Note that this paper is almost a year old and was written before the DODS Java software was itself written. It contains some errors.
