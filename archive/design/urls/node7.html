URLs and the Data Locator

The Data Locator is used to find specific data sets within DODS. Users can query the Data Locator and expect that it will return a URL which can be used by other components in DODS to access that data set. These URLs may be used as supplied to user programs or they may be edited first. Because the URLs are ASCII text strings, no special software is required to edit or store them. In addition, DODS URLs may be obtained using other methods such as Mosaic, e-mail, or personal communications.
