XDODS-Server

The XDODS-Server header is used to return DAP server's implementation version information to the client program.5 This header MUST be included in every response.

XDODS-Server = "XDODS-Server" ":" "dods/" version
version = DIGIT "." DIGIT [ "." DIGIT ]

Example: 

XDODS-Server: dods/3.2.2
