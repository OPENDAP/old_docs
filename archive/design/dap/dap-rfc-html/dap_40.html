Help

The help response MUST be returned when either the server receives a URL with no extension (i.e., a URL which asks for no object) or when the dataset_id portion of the URL is help.

abs_path = server_path dataset_id "." ext [ "?" query ]
server_path = <name of DAP server>
dataset_id = "help"

The second way of requesting the help response is analogous to one of the ways of requesting the version response.

The help response MUST return an HTML document which lists the extensions recognized by the server. The response MAY return other information as well.
