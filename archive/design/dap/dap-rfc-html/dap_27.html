Keep-Alive

A DAP server (or an underlying HTTP server if one is used to implement the DAP server) MAY return a Keep-Alive header for an authenticate (code 401) response. For all other responses, the DAP server MUST NOT return this header.4 See RFC 2616 [4] for this header's grammar.
