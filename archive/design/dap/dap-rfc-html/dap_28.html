Server

The Server header provides information about the server used to process the request. In this case the server MAY be either the DAP server or an underlying HTTP server if the DAP server uses that as part of its implementation. See RFC 2616 [4] for this header's grammar. This header is OPTIONAL.

Example: 

Server: Apache/1.3.12 (Unix)  (Red Hat/Linux) PHP/3.0.15 mod\_perl/1.21
