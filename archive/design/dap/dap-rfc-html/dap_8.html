List

The List type constructor is used to hold lists of 0 or more items of one type. Lists of byte, ..., grid are specified using the keyword list before the variable's class. The list can be of any type except List.

list-decl = "list" (simple-decl | array-decl | structure-decl
| sequence-decl | grid-decl)

Examples: 

list int32 heights;
List Float64 x[10][10];
