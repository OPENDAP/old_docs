A Summary of the Distributed Oceanographic Data System

Three component types comprise DODS:
  • Core classes used to model and contain data
  • Storage-format independent data servers
  • Existing APIs extended to read from DODS data servers






An over-arching goal of DODS is to meet the scientist on their terms rather than require them to meet our requirements. Data can be served using DODS without changing its local storage form, existing software can be used to access those data and new software is straightforward to write.
