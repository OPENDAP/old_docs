Breadth of data types

Analysis software:
  • Must expect to find an infinite number of scientific data formats (or types)
  • Must be able to process metadata as data
  • Only process a small set of data structures






In order to exploit the structural similarity inherent in most scientific data, analysis software must be able to use metadata as data. This provides a way to build a highly inter-operable data system.
