Summary

In the transition from a closed to a distributed system:
  • Searching must be closely coupled with analysis and must be iterative

  • Data servers must make more metadata accessible as data

  • Analysis programs must convert data from `network' structures to `science' data types for local use

  • Processing data must be partially off-loaded to the server
