Characteristics of remote data

Three characteristics drive the requirements of analysis software:
  • Data volume
  • Breadth of data types
  • Sharing versus broadcast of data






Shifting from local data to remote data adds new requirements to analysis programs since the difference in data volume and type are potentially so great that existing techniques for handling these differences fail to meet user's demands for flexibility.
