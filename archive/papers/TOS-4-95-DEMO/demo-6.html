DODS Client libraries:

  • Can be used by many programs - The time spent modifying the original library is justified by the large number of user programs which can benefit from the new library

  • Facilitate the modification of legacy software

  • Simplify the development of new network-aware software

  • Enable analysis packages to read data stored in many formats










Because DODS client libraries communicate with DODS servers using a single language, they can read data which is stored in many different formats.
