Figure 1a: The freely available program `NCview' after being rebuilt with the DODS-netCDF library. The rebuilt program is run exactly the same as the original, except that a URL specifying the remote data set is used in place of a file name.

Figure 1c: The data appears no differently than a local file.





Several programs have been rebuilt using alpha versions of both the DODS-netCDF and DODS-JGOFS libraries (with no changes to the application source code).
