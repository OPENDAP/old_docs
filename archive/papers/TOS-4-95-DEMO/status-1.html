Status of DODS Software from URI/MIT:

  • The second version of the core software has been released.

  • Four general-purpose data servers have been developed.

  • Two client-libraries have been released.

  • Version 2 has been built on four platforms (Sun Sparc 4.1.3, Sun Sparc Solaris, Dec Alpha and SGI).

  • Design and basic prototyping has begun on the locator.






The second version of DODS combines the basic transport features of version 1 with a powerful system for controling the data servers. Additional data servers have been developed, and the short time required to build these new components illustrates that other servers can be completed quickly.
