The Data Access Protocol

There are three messages in the DAP:

  • The Dataset Descriptor Structure contains information about the names, types, etc. of variables in a data set.
  • The Dataset Attribute Structure contains ancillary information about the dataset and its components.
  • A data message contains binary information.




The Data Access Protocol uses a type definition language to communicate information about the contents of a data set.
