Availability of Data

Previously:

  • Local data only
  • Minimal sharing of data
  • Researcher was intimately familiar with the data and its structure
  • Data was formatted specially for the researcher's data analysis applications
  • Researcher's data system was customized to provide efficiency or flexibility for specific types of data

Now:

  • Both remote and local data
  • Shared data comes from many other scientists and centralized archives
  • Global (community resource) datasets are being generated for wide distribution
  • Data exists in many different structures which are dependent on source and content
  • Researchers continue to use their own scientific applications for analyzing data
