Using DODS to Access and Deliver Remote Data

5. How To Get Help

Several resources are available if you run into difficulties. The DODS web site

    http://www.unidata.ucar.edu/packages/dods/

contains user documentation and support resources. Besides the documentation, the support resources include an FAQ page and searchable support e-mail archives.

For questions, please e-mail us at support@unidata.ucar.edu.
