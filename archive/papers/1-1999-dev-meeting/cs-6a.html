Standard variables and query interfaces

  • Implemented functions for standard variables:
    DODS_Date(container)
DODS_Time(container)
DODS_Date_Time(container)

  • Implemented functions for queries:
    date(date string 1[, date string 2])
time(time string 1[, time string 2])
date_time(date/time string 1[, date/time string 2])

  • Examples:
    DDS:                         DAS:
Dataset {                    Attributes {
    Sequence {                   DODS_Global {
        String DODS_URL;             String DODS_Interfaces date, time;
	Int32 year;            	     Int32 DODS_Standards 1;
	Int32 year_day;          }
	Int32 h;                 DODS_Time {
	Int32 m;                     ...
	Float64 s;               }
    } URI_Avhrr;                 DODS_Date {
} URI_Avhrr;                         String year_variable year;
	                             String year_day_variable year_day;
	                             Int32 year_base 1900;
                                 }
                                 ...
                             }
