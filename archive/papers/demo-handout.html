The Distributed Ocenographic Data System
Demonstration, Winter 1995


About DODS:

  • Why:
    • Making data available should be as easy as writing it in a file.
    • Data should be directly accessible by analysis applications.
  • What:
    • DODS provides network access to scientific data.
    • Clients are made using existing analysis programs.
    • Data servers are built using httpd.
    • Data is not modified.
    • Servers access data in several formats or are customized to `in house' formats.
  • Where:
    • WWW Home Page: http://dods.gso.uri.edu/
    • Software: ftp://dods.gso.uri.edu/pub/dods/
  • Who:
    • The University of Rhode Island and the Massachusetts Institute of Technology: Design, Development, Management.
    • The Oceanography Society: Workshop managment support.
    • The Jet Propulsion Laboratory: Development of HDF components.
    • NOAA NODC: Access to national data archives.
    • Unidata, inc., NOAA PMEL: Advisors.
    • Many others have provided advice on DODS.
