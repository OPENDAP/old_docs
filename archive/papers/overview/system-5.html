Client programs are built using existing software.

  • A program which uses a supported API to read local data can be rebuilt to read data from DODS servers.

  • The source code which implements the API is modified, not the user program - Most user programs can be rebuilt with no re-programming.

  • Linking analysis programs with DODS libraries means that users can use existing analysis packages/procedures (including those they have developed in-house) rather than picking from a handful of client programs developed specifically for DODS.
