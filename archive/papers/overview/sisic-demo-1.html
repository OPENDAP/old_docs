Demonstrations of DODS

  • Data
    • Data from the NODC NODB - The server uses the DODS-JGOFS data server software and a new JGOFS method.
    • NetCDF data from URI - the server uses the DODS-NetCDF server software.
  • Client programs
    • MatLAB - The MatLAB client uses the DODS-JGOFS client library and the JGOFS interface to MatLAB.
    • NCView - The NCView client uses the DODS-NetCDF client library and the NCView application.





DODS servers do not require that data be modified; data stored in a supported format can be served by installing a pre-built server while other data requires a custom server. Client programs are made by relinking existing application software or can be built from scratch using the core software.
