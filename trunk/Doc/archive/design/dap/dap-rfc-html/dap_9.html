Array

An Array is a one dimensional indexed data structure as defined by ANSI C@. Multidimensional arrays are defined as arrays of arrays. The size of each array's dimensions must be given. Each dimension of an array may also be named.

Multi-dimensional arrays are stored in row-major order (as is the case with ANSI Cor C++). Any array stored in row-major order is stored so that the last dimension varies most quickly. For example suppose the two dimensional array String letter[2][3] has two rows of three columns each and looks like:

A B C
D E F
the values would be stored as A B C D E F in memory.

array-decl = array-types id array-dims ";"
array-types = simple-decl | structure-decl | sequence-decl | grid-decl
array-dims = array-dim | array-dim array-dims
array-dim = "[" [ name "=" ] DIGIT *DIGIT "]"
