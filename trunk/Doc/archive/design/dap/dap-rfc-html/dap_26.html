Date

The Date header provides a time stamp for the response. This header is needed for servers that support caching. See RFC 2616 [4] for this header's grammar. Servers MUST provide this header.

Example: 

Date: Fri, 09 Feb 2001 18:54:55 GMT
James Gallagher <jgallagher@gso.uri.edu>, Date: 2001/09/18 18:28:50 , Revision: 1.18
