The HTML response MUST present the user with an HTML document that can be used to access values from the dataset. The document MUST show all of the variables in the dataset along with their attributes and it MUST show the dataset's attributes. Structural information SHOULD be preserved.

The intent of the HTML response it to provide an interface to the dataset that can be used with a WWW browser. Users should be able to look at the variables in the dataset, read attributes and request values from one WWW location without making multiple requests.

Because DAP URLs are somewhat confusing for many people, this interface should provide a point-and-click way to build those URLs.
