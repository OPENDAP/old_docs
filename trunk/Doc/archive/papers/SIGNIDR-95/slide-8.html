APIs as Transport Mechanisms

While an API makes a good transport mechanism from some perspectives, it has significant shortfalls as well:

  • Each API defines its own non-standard communication protocol.
  • Interoperation between clients using API X and servers using API Y is hard.
  • APIs may not deliver acceptable performance on a wide area network.




We would like a standard transport mechanism which retained the benefits of the simpler API-based design
