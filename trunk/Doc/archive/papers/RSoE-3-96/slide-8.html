Sharing data

Sharing data between individuals is the most important mechanism for data exchange:
  • All data consumers are potential providers
  • Individual providers must never be `second-class' citizens in a system
  • Data made accessible to remote sites must be stored so that it can be used locally too






Earth Scientists repeatedly say that the most important external source of information comes from their colleges. Thus any distributed data system must provide access to this information if it is to radically alter researcher's access to data.
