Data servers compliant with the DODS DAP:

  • NetCDF - Version 1 and 2

  • JGOFS - Version 1 and 2

  • FreeForm - Version 1

  • CDF - Version 1 (developed at UMD)

  • HDF - Version 2 (preliminary version, developed at JPL)




The NetCDF and JGOFS data servers released with the DODS version 1 core software have been updated to work with version 2. A data server for the FreeForm API was built using the version 1 software but has not yet been updated to work with version 2. Groups at the University of Maryland and the Jet Propulsion Laboratory have built servers for CDF (works with DODS version 1) and HDF (an ongoing effort; works with DODS version 2).
