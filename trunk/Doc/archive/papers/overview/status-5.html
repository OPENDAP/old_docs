Other groups working on DODS:

  • At the Jet Propulsion Laboratory: Work on a HDF client-library and data server are progressing.

  • At the University of Maryland: Work on accessing space-science data was completed (6/96). This resulted in a version 1 data server for the CDF API and client software for IDL being built.

  • At NASA Goddard Space Flight Center: Work on integrating DODS data sets into the NASA Global Change Master Directory continues.







The URI/MIT group encourages other groups to comment on, and participate in, the design and development of DODS. We see this as one of the most important aspects of the project.
