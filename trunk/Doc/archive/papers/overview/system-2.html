Data are accessible through a server:

  • Each dataset has an associated server

  • Data are distributed throughout computers on the Internet

  • Data may be located at many different types of sites including national archive centers and individual researcher's computers











A system that places essentially no restriction on the data it makes available raises a number of issues about data validity. However, this problem is a network-wide problem which must be addressed in a comprehensive fashion.
