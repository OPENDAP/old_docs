Data retain their format:

  • Servers are customized to data, not vice-versa

  • All servers are accessed using a single protocol

  • Data are accessed through servers which facilitate the translation from the DODS data access protocol to the local format for the data











Using a single protocol to access all data potentially enables any program to access all of the data that is part of the system. This feature complicates building the clients and servers, but provides a much more powerful system for the user.
